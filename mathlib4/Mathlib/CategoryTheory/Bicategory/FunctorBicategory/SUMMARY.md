# FunctorBicategory

## Overview

The `FunctorBicategory/` directory contains the bicategory structure on oplax functors between bicategories. It defines a bicategory `B ⥤ᵒᵖᴸ C` where objects are oplax functors, 1-morphisms are oplax natural transformations, and 2-morphisms are modifications, with the necessary whiskering operations, associators, and unitors.
