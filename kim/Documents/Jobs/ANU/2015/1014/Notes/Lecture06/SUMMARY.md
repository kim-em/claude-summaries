# Lecture06

## Overview

Sixth lecture on null spaces, column spaces, and linear transformations in MATH1014. The main lecture introduces the null space and column space of matrices as fundamental subspaces, proves that the null space is a subspace of ℝⁿ and the column space is a subspace of ℝᵐ, and generalizes these concepts to abstract vector spaces through linear transformations (defining kernel and range). The examples file provides detailed worked problems computing kernels and ranges for various linear transformations between abstract vector spaces (polynomials, matrices).
