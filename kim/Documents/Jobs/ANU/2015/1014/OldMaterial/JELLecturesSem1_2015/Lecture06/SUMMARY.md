# Lecture06

## Overview

Lecture 6 materials covering null spaces, column spaces, and linear transformations - the transition from matrix-specific concepts to abstract vector spaces. The lecture introduces how matrices define linear transformations between vector spaces, generalizing null space to kernel and column space to range. Includes extensive examples of finding kernels and ranges for transformations between polynomial spaces, matrix spaces, and R^n. Two versions of the main lecture file exist (Lecture06.tex and Lecture06actual.tex), with the "actual" version including additional warm-up examples on subspaces. Supplementary handouts compare null/column space properties and provide worked examples for various linear transformations.
