# Lecture03

## Overview

Lecture materials for Lectures 3 and 4 of MATH1014, covering vector geometry in R² and R³. This folder contains LaTeX source files and Beamer presentations teaching lines and planes using vector equations, parametric equations, and symmetric equations. The content includes the introduction of the cross product and applications to computing distances between geometric objects. Multiple versions exist: the original Lecture03.tex, a 2015 revision (Lecture03.2015.tex), and Lecture04.tex which covers additional distance formulas. The folder includes compiled PDFs, Graffle diagram source files, and supporting style files.
