# EssayExamples

## Overview

Example essays demonstrating expected quality and format for the MATH3325 Analysis 3 Honours essay component (Semester 2, 2014). Contains four student essays (6-11 pages each, 206-228KB) covering advanced topics in functional analysis and representation theory: trace class operators and Lidskii's Theorem (Tom Phelan, 2009), the Peter-Weyl Theorem on compact Lie groups (Ben Lim, 2013), Bochner's Theorem on Fourier transforms (Yitao Lei, 2013), and fixed point theorems including Markov-Kakutani and Kakutani theorems (Hannah Keese, 2013). Essays illustrate appropriate mathematical exposition style, proof techniques, and breadth expected from students.
