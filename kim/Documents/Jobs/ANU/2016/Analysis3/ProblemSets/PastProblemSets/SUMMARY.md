# PastProblemSets

## Overview

Archive of problem sets from earlier offerings of MATH3325 Analysis 3, specifically from 2010 and 2012 vintages. Contains five LaTeX source files covering advanced topics in measure theory, functional analysis, and weak topology that were not included in the 2016 course materials. These appear to be source files only (no compiled PDFs or solutions included), representing different pedagogical approaches and topic selections from previous years.

## Key Files

| File | Purpose |
|------|---------|
| `probset4.tex` | Problem Set 4 from 2012 (Sept 3 tutorial): 4 problems on measure theory including measure defined by integration with respect to another measure, translation-invariant measures on R^n and uniqueness of Lebesgue measure, independence of simple function integral representation, and absolute continuity examples |
| `probset5.tex` | Problem Set 5 from 2012 (Oct 3 tutorial): 4 problems on measure theory including measurable sets occurring infinitely often, absolutely continuous functions and measures, singular measures without atoms, and Radon-Nikodym theorem counterexamples with counting measure |
