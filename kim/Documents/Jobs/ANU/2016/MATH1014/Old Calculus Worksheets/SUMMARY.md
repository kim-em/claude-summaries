# Old Calculus Worksheets

## Overview

Archive of calculus tutorial worksheets from previous years (2013 Semester II, 2014 Semester II, and 2015 Semester I) for MATH1014. These historical materials include weekly problem sets covering topics such as Riemann sums, improper integrals, convergence/divergence tests, integration by parts, and other first-year calculus topics. Each semester's collection contains 10 weekly worksheets with LaTeX source files and solutions. The 2014-15 collections also include compiled PDF versions combining all worksheets.
