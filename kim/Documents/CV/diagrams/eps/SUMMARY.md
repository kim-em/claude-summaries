# eps

## Overview

Encapsulated PostScript (EPS) diagram files for Temperley-Lieb algebra and basic knot diagram primitives. Contains five numbered Temperley-Lieb diagrams (TL-2-4-1 through TL-2-4-5) and three basic diagram building blocks (crossing, horizontal, vertical). All files are vector graphics in PostScript format generated by Ghostscript, designed for inclusion in LaTeX documents. These are the source vector files that can be compiled or converted to other formats using the scripts in the parent directory.
