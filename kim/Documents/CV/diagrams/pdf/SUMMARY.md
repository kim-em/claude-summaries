# pdf

## Overview

PDF versions of mathematical diagrams for Temperley-Lieb algebra and knot theory, corresponding to the EPS versions in the sibling directory. Contains five Temperley-Lieb diagrams (TL-2-4-1 through TL-2-4-5) showing various cup and strand configurations, three primitive diagram components (crossing, horizontal caps, vertical caps), and a handwritten signature image in JPEG format. These PDFs serve as the output format for LaTeX document inclusion and can be converted to EPS format using the scripts in the parent's scripts directory.
