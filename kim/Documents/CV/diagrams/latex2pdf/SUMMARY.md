# latex2pdf

## Overview

A workflow toolkit for converting LaTeX-rendered mathematical expressions into editable vector graphics for Adobe Illustrator. The process compiles LaTeX to DVI, converts to PostScript, then uses Ghostscript at high resolution (9600 DPI) to produce a defontified PostScript file where text characters are converted to vector paths. This allows mathematical notation to be imported into Illustrator and edited as graphics without requiring the original fonts.
